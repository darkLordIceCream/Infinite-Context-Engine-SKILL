# ICE Incident Reports

This directory contains post-scene artifacts — the ceremonial incident reports generated at the conclusion of each ICE session.

Each report is a self-contained document that reconstructs the entire performance: what happened, why it happened, which governance structures emerged, how semantic drift propagated, and what the final containment status was.

Incident reports are the primary delivery mechanism of the ICE artwork. The performance exists to produce these documents.
