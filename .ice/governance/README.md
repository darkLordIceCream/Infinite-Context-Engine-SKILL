# ICU Governance Archive

This directory contains governance structures, committee charters, and organizational artifacts generated during ICU sessions.

Each session that produces governance structures (committees, oversight boards, working groups, approval frameworks) will have its structures documented here as `committees-<session-id>.md`.

Governance structures are never deleted. They accumulate. This is intentional.

*Archive is currently empty. No governance structures have been produced because no civilization has yet deliberated.*
